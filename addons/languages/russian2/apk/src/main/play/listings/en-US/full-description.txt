This is a Cyrillic expansion pack for AnySoftKeyboard.
Install AnySoftKeyboard first, and then select the desired layout from AnySoftKeyboard's Settings->Keyboards menu.

This pack contains Cyrillic and phonetic Russian keyboard layouts, physical keyboard mapping and dictionary.

Search for AnySoftKeyboard on Google Play.
